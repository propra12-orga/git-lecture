\section{Github}
\subsection*{Einleitung}
\begin{frame}
  \frametitle{Github}
  \begin{itemize}
    \item Kostenlose Öffentliche GIT-Repositories
    \item Arbeit in Teams an Projekten möglich
    \item Authentifizierung mit Public Key
  \end{itemize}
\end{frame}
\subsection*{Projektarbeit}
\begin{frame}
  \frametitle{Projektarbeit}
  \begin{enumerate}
    \item Account anmelden
    \item SSH-Schlüssel erstellen
    \item Public-Key bei Github hinterlegen
    \item Auf die Übung nächste Woche vorbereitet sein
  \end{enumerate}
\end{frame}
%
\begin{frame}

    Bis spätestens Freitag 20. April 2012 (Morgen) um 7.00 Uhr morgens in Gruppen eingefunden haben und nach Möglichkeit weitere Alternativtermine angeben!\\
    Leute die keine Gruppe gefunden haben werden zufällig eingeteilt. In den ersten Übungen nächste Woche wird GIT eingerichtet und das Projekt gestartet.
  \end{block}
\end{frame}
